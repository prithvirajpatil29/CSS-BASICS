Defining color value

Colors in css most often specifies in the following formats:

  1. A color keyword - like - "red","green" etc
  2. A hex value - like "#ffoooo", "00ff00", etc
  3. A RGB values(Red, Green, Blue) - like "rgb(255,0,0)"
